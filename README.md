# GitHub Learning Lab 🎓

Welcome to your interactive GitHub learning experience! This repo is designed to help you master **Pull Requests**, **Forking**, and **Collaborative Workflows**.

## What You'll Learn

### Level 1: Pull Requests (PRs)
- What PRs are and why they matter
- Creating a PR from a branch
- Reviewing and merging PRs
- PR best practices

### Level 2: Forking Workflow
- When and why to fork
- Forking a repo to your account
- Syncing forks with upstream
- Creating PRs from forks

### Level 3: Real Collaboration
- Code review etiquette
- Handling review feedback
- Resolving merge conflicts
- Branch protection rules

## Quick Start

```bash
# Clone this repo
git clone https://github.com/yaghta/github-learning-lab.git
cd github-learning-lab

# Follow the exercises in order:
# 1. exercises/01-pull-request-basics/
# 2. exercises/02-forking-workflow/
# 3. exercises/03-collaboration-patterns/
```
